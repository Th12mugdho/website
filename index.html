Selected projects

ROS2 · PX4 Final Year Design Project

Payra — Autonomous Delivery Hexacopter

A hexacopter-based non-landing drone delivery system for urban food and medicine delivery in Dhaka, with an IMU-driven active fan stabilization approach for the payload box.

Pixhawk flight controller ESP32 smart delivery box Gazebo SITL simulation
Bio-inspired locomotion

MechaHex — Six-Legged Hexapod

A six-legged hexapod robot navigating terrain via a tripod gait, with real-time motor coordination and trajectory planning for stable walking, turning, and obstacle avoidance.

Arduino control Custom 3D-printed chassis Ultrasonic sensing
Embedded vision

Live Surveillance Node

A compact surveillance system built on a Raspberry Pi Zero 2W with a Flask-based web app for real-time video streaming and remote monitoring.

Raspberry Pi Zero 2W Flask streaming server

Payra — FYDP Design Report

Formal design documentation covering system specifications, requirements, constraints, and conflicting-requirement trade-off tables for the delivery drone platform.

Requirements engineering URDF/xacro modeling

Active Filter Design Suite

Transfer function derivation, Bode plot analysis, and active filter design using Multiple Feedback (MFB) and Twin-T topologies, visualized with interactive Chart.js tooling.

MFB / Twin-T Bode analysis
VLSI

CMOS Delay & Sizing Study

Elmore delay modeling and transistor sizing across pull-up/pull-down networks, paired with Boolean minimization via Karnaugh maps for digital logic design.

Elmore delay PDN / PUN sizing
Applied physics research
